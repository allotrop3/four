Class: Camera

Entity. Camera

new Camera(nameopt, program, pathopt, uniformsopt, backgroundopt)

A camera simplifies transforming the perspective of the view, including scaling, rotating and translating, thereby configuring projections into framebuffers or output display.
Parameters:
Name Type Attributes Default Description
name string <optional>
 camera Specifies the entities friendly name.
program string Specifies the program in which the camera used.
path string <optional>
 camera Specifies the uniform structure path to the generic uniforms.
uniforms Array <optional>
 ['mat4 projectionMatrix', 'mat4 modelViewMatrix', 'mat3 normalMatrix'] Specifies the formats and names of the generic uniforms as used in the shader.
background vec4 <optional>
 [0, 0, 0, 1] Specifies the clear value for the color buffers.
