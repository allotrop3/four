Class: OBJMeshLoader

Entity.MeshLoader. OBJMeshLoader

new OBJMeshLoader(nameopt, path)

An OBJ mesh loader is a .obj file parser that extracts the collections of vertices and faces that define a shape. It may also include data to shade the mesh, such as vertex colors, coordinates and per-vertex/face normals.
