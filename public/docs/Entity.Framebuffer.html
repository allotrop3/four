Class: Framebuffer

Entity. Framebuffer

new Framebuffer(nameopt)

A framebuffer is a collection of buffers that can be used as the destination for rendering. There are two kinds of framebuffers: the default framebuffer, which refers to the canvas element, and user-created framebuffers called framebuffer objects - see Entity.Framebuffer.DeferredFramebuffer.
