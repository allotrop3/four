Class: Shader

Entity. Shader

new Shader(nameopt, path, type)

A shader is used to maintain the source code strings that define a shader. Two types of shader are supported. A shader of type gl.VERTEX_SHADER is a shader that is intended to run on the programmable vertex processor. A shader of type gl.FRAGMENT_SHADER is a shader that is intended to run on the programmable fragment processor.
