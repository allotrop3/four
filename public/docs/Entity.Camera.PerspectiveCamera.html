Class: PerspectiveCamera

Entity.Camera. PerspectiveCamera

new PerspectiveCamera(nameopt, program, pathopt, uniformsopt, backgroundopt, widthopt, heightopt, fovopt, nearopt, faropt, directionopt, location, upopt)

An perspective camera produces a three dimensional projection into framebuffers.
Parameters:
Name Type Attributes Default Description
name string <optional>
 perspective.camera Specifies the entities friendly name.
program Entity.Program Specifies the program in which the camera used.
path string <optional>
 camera Specifies the uniform structure path to the generic uniforms.
uniforms Array <optional>
 ['mat4 projectionMatrix', 'mat4 modelViewMatrix', 'mat3 normalMatrix'] Specifies the formats and names of the generic uniforms as used in the shader.
background vec4 <optional>
 [0, 0, 0, 1] Specifies the clear value for the color buffers.

 gl.canvas.width Specifies the projection width. The initial value is set to the width of the canvas.

 gl.canvas.height Specifies the projection height. The initial value is set to the height of the canvas.

 45 Specifies the field of view of the projection.

 0.1 Specifies the front most boundary of the projection.

 1000 Specifies the back most boundary of the projection.

 [0, 0, 0] Specifies the position at which the projection is aimed.
