Class: SpotLight

Entity.Structure.Light. SpotLight

new SpotLight(nameopt, program, pathopt, uniformsopt, ambient, diffuse, specular, exponent, cutoff, intensityopt, locationopt, directionopt)

A spot light defines a directional cone-shaped radiation of light to illuminate a specific area.

Parameters:
Name Type Attributes Default Description
name string <optional>
 spot.light

Specifies the entities friendly name.

Specifies the program in which the uniforms should be located.

 light

Specifies the shader struct path to the uniforms, if required.

 ['vec3 ambient', 'vec3 diffuse', 'vec3 specular', 'f exponent', 'f cutoff', 'f intensity', 'vec3 location', 'vec3 direction']

Specifies the uniform variables as is declared in the shader. The names imply the u_ namespace is used as per the uniform variable naming convention. If a struct path is defined, the u_ namespace is implied on this instead.
