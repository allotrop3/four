Class: MeshLoader

Entity. MeshLoader

new MeshLoader(nameopt, path)

A mesh loader is a mesh file parser that extracts the collections of vertices and faces that define a shape. It may also include data to shade the mesh, such as vertex colors, coordinates and per-vertex/face normals, depending on the mesh file format.
