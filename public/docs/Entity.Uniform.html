Class: Uniform

Entity. Uniform

new Uniform(nameopt, pathopt, program, uniform, format)

Uniform is a wrapper on shader WebGLUniformLocation uniforms. Supported types include i, vec2i, vec3i, vec4i, f, vec2f, vec3f vec4f, mat2, mat3, and mat4
